# OAC_II
Repositório para a disciplina de OAC II

### Instalação Linux Ubuntu/Mint
`sudo apt update`
`sudo apt install iverilog`

### Execução
`iverilog -o <nome_exec> <arquivos.v>`

`vvp <nome_exec>`

### Instalção do GTKWave
`sudo apt update`
`sudo apt install gtkwave`

### Execução GTKWave

`gtkwave <arquivo>.vcd`
